# Event release

This a event release system created by using React for frontend and Gin (Golang) framework for backend. This repository is created based on the repository called RGB
on GitHub, but added many other features for event release activity. 

Please watch the video clip demo.webm to see what it works.

Tips:
1. On Windows run scripts/deploy.bat to start the server.
2. To start the client, go to assets directory and run npm start.
